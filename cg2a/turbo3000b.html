Utilisation des boutons poussoir
Exploitation menu 00 a 07 pour les versions V39 et V40
T1
 Permet de choisir le menu dans lequel on désire se rendre, 8 menu sont accessibles par appuis successifs
T2
 Permet de sélectionner le code "LIT"(lecture) ou "CHAN"(changement) ainsi que sélectionner les afficheurs AFF1 - AFF2 ou AFF3 - AFF4
T3
 Permet d'incrémenter (augmenter) les valeurs sélectionnées dans le menu choisi en mode "CHAN"
T4
 Permet de décrémenter (diminuer) les valeurs sélectionnées dans le menu choisi en mode "CHAN"
Exploitation menu 08 a 147 pour les versions V41, V42 et V43
T1
 Permet de choisir le menu dans lequel on désire se rendre, 14 menu sont accessibles par appuis successifs
T2
 Permet de sélectionner le code "LIT"(lecture) ou "CHAN"(changement) .Permet de passer du groupe AFF1 - AFF2 clignotant au groupe AFF3 - AFF4 clignotant et inversement
T3
 Permet en général de d'incrémenter (augmenter de 1) les valeurs affichées sur AFF3 ou AFF1 (clignotant)
T4
 Permet en général de d'incrémenter (augmenter de 1) les valeurs affichées sur AFF4 ou AFF2 (clignotant)

Lecture des défauts en mémoire ce trouve au MENU 05 (affichage des 16 derniers défauts) , les afficheurs AFF1-AFF2 vous indiquent le numéro d'ordre de défaut (de01 a 16) et l'afficheur AFF3-AFF4 le code de défaut. Chaque défaut est afficher durant 2 sec.

Pour effacer toute la mémorisation des codes de défauts maintenez la touche T4 appuyée pendant 4 sec , l'abréviation "EFFA" est alors affiche pendant ce temps
