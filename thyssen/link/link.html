---------------

Paramètre:Accès

Ce paramètre permet d'attribuer
le tableau de bord à l'accès correspondant

* 1 Pour l'accès principal

* 1bis Pour le 2ème tableau de bord de l'accès principal

* 2 Pour l'accès secondaire

* 2bis Pour le 2ème tableau de bord de l'accès secondaire

-----

Vérifier si le paramètre Accès est correcte

Si OUI,accéder au paramètre TVal
avec les touches
  

Si NON,modifier le contenu du paramètre Accès
avec les touches
  

Valider avec la touche:

---------------

Paramètre:Temps de Validation et Temps

Temps de validation:

Ce paramètre permet d'ajuster,si nécessaire,
le temps de validation de l'envoi.Par défaut,ce temps est de 1sec
(Ajustable par pas de 0,2sec max 5sec)

-----

Vérifier si le paramètre TVal est correcte

Si OUI,accéder au paramètre T-
avec les touches
  

Si NON,modifier le contenu du paramètre TVal
avec les touches
  

Valider avec la touche:

---------------

Temps - :

Ce paramètre permet d'ajuster,si nécessaire,
le temps de saisie maximun d'une touche(1 à 9)
aprés activation de la touche    .
Par défaut ce temps est de 2sec
(Ajustable par pas de 1sec mini 2sec/max 5sec)

-----

Vérifier si le paramètre T- est correcte

Si OUI,accéder au paramètre Gong
avec les touches
  

Si NON,modifier le contenu du paramètre T-
avec les touches
  

Valider avec la touche:

---------------

Paramètre:Gong et Synthèse

Ces paramètre permettent ,si nécessaire,d'adapter le type

de signal sonore diffusé en cabine
lors de l'arrivée à un niveau

-----

Par défaut:Gong:Non-Synthése:Oui

Vérifier si le paramètre Gong est correcte

Si OUI,accéder au paramètre Synth
avec les touches
  

Si NON modifier le contenue du paramètre Gong avec la touche

Cette action permet de valider,modifier et

de passer au paramètre suivant.

---------------

Vérifier si le paramètre Synth est correcte

Si OUI,accéder au paramètre Code
avec les touches
  

Si NON modifier le contenue du paramètre Synth avec la touche

Cette action permet de valider,modifier et

de passer au paramètre suivant.

---------------

Paramètre Codes

Ce paramètre permet si besoin

de personnaliser le code d'entrée:
Du menu Paramètrage.Par défaut ce code est 9012

D'usage cabine seule PRIC.Par défaut ce code est 1234

D'actiation/désactivation ventilation cabine.Par défaut ce code est 5678
