DHCS (dispositif  hors course sécurité)

Dispositif hors course sécurité (DHCS)

Le déclenchement de ce dispositif bloque le fonctionnement de l'appareil et crée l'affichage du code erreur n°24

L'apparition de cet évènement est mémorise, garde en mémoire, même en cas de coupure de l'alimentation général.

POUR EFFACER LA MEMOIRE DE L'APPAREIL, VOUS DEVEZ

- Appuyer simultanément sur les deux boutons poussoir  et  jusqu’à apparition a l'afficheur du code EF , puis du code 00

N.B. En mode inspection ce défaut est neutralise
