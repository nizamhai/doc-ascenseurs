----------------
Symptômes
----------------

1.0	La Led 12 (EMLS4) est allumée ou clignote.
	
2.0	Le Led 27 (POT) peut être allumée.

-------------------------
Causes Probables
--------------------------

1.0	La tension de manoeuvre est incorrecte.

2.0	Le système constate que:

	2.1	Le frein n'est pas retombé (contacts BS).

	2.2	Le relais EES n'est pas retombé.

3.0	Le fonctionnement du relais EES est incorrect.

4.0	Le fonctionnement des relais C / CX est incorrect.

5.0	Le fonctionnement du contacteur H est incorrect.

6.0	Le fonctionnement des contacteurs 1M / 2M
	(ou 2CR / 3CR) est défectueux.

7.0	Le fonctionnement du contacteur BPS est incorrect.

8.0	Le relais MC (excitation) ou PSO (alimentation CEPS)
	n'est pas collé.

9.0	Le relais SSR (contrôle du ralentissement aux extrêmes)
	n'est pas collé à l'arrêt.

-------------
Remèdes
-------------
	  
Nota:	Le système contrôle la séquence suivante:

	- Le frein doit être retombé (/BS=1).

	- Le relais EES doit être retombé (EES=0) quand l'ordre 
	  de collage est nul (EESE=0).

	- Le relais EES doit collé (EES=1) quand l'ordre est donné 
	  (EESE=1).

	- Au niveau du câblage, le collage de EES permet aux relais
	  C et CX de coller et de s'auto-maintenir si :

		- MC et PSO sont collé.
		- SSR est collé (retombe à 90% Vitesse Nominale)
		- MFS est collé (contact MFS 11/12).
		- Le relais ESC (arrêt d'urgence) est retombé.
		- Les contacteurs 1M / 2M (et leur relais 2CR / 3CR)
		  sont retombés.
		- Les contacteurs H et BPS sont retombés.

	- Au niveau du câblage, le collage de C ramène du 110 VDC 
	  sur la bobine H (sans la faire coller).

	- Le contacteur H doit collé (/H=0) quand l'ordre est donné 
	  (HESS=1).

	- L'ordre de marche est donné au convertisseur (DDRUN=1).

	- Le convertisseur doit collé le relais 1CR qui fait à son tour 
	  coller 1M / 2M (et 2CR / 3CR).

	Suivant les versions de logiciel:

	- Le contacteur BPS doit collé (BPS=1) quand l'ordre est donné
	  (BPSE=1).

	Lors du passage en MLS4, une temporisation (TS4) est remise
	à 0. Si les conditions décrites ci-dessus ne sont pas atteintes
	avant le temps maximal (TS4MAX = 50 Hex # 8 sec) alors le 
	défaut EMSL4 est actif.

1.0	Vérifier:

	1.1	La tension de manoeuvre sur la borne A8 TB1-24 doit
		être de 110 vdc:

		1.1.1	Vérifier le condensateur C1.

		1.1.2	La chute de tension dans les serrures.

		1.1.3	Le relais MC doit être collé (excitation).

			1.1.3.1	Vérifier les connections du circuit
				d'excitation, du transformateur au
				moteur (cosses fondues ou fils
				desserrés).

			1.1.3.2	Vérifier les tensions + 8 et +18 vdc
				fournies par l'alimentation CEPS à
				l'ampli MF/BK.

			1.1.3.3	Vérifier que le connecteur A4-J2 de 
				l'ampli soit correctement inséré dans
				son support.

			1.1.3.4	Vérifier l'ampli MF/BK (OMT Test 250).

			1.1.3.5	Remplacer l'ampli MF/BK.

		1.1.4	Le relais PSO doit être collé (Alim. CEPS).
		
			1.1.4.1	Vérifier toutes les tensions (CEPS).

			1.1.4.2	Vérifier le serrage des connecteurs
				A1-J1 et A1-J2.

			1.1.4.3	Remplacer l'alimentation CEPS.

		1.1.5	Le relais SSR doit être collé à l'arrêt.

			1.1.5.1	Vérifier la tension AC3.

			1.1.5.2	Vérifier la bobine et l'état du relais.

			1.1.5.3	Remplacer la carte GP I/O (Slot 7).

	Nota:	Pour tous les EMLS(x), la tension de manoeuvre 
		peut être la cause du défaut si elle descend sous
		les 100 vdc aux bornes des contacteurs.

2.0	Vérifier:

	2.1	/BS à l'adresse @ F2DE doit être à 1 (contact rupture).

		2.1.1	La tension sur la borne A6 TB1-57 doit être de
			110 vdc.

		2.1.2	Remplacer la carte CF I/O.

	2.2	Le relais EES doit être retombé quand EESE (@ F3EC)
		est à 0.

		2.2.1	Rechercher une masse éventuelle.

		2.2.2	Vérifier s'il existe un dur mécanique ou si un 
			contact est soudé.

		2.2.3	Vérifier si le relais n'est pas rémanent.

		2.2.4	Relais retombé, le signal EES (@ F358)
			doit être nul.

			2.2.4.1	La tension en A6 TB1-54 doit être
				de 110 vdc.
			2.2.4.2	Vérifier le contact avant la borne 
				A6 TB1-54.
			2.2.4.3	Remplacer la carte CF I/O.

3.0	Le relais EES doit collé quand EESE (@ F3EC) est à 1.

	3.1	Vérifier la bobine.

	3.2	Vérifier s'il existe un dur mécanique.

	3.3	Remplacer la carte Relay I/O.

	3.4	Relais collé, le signal EES (@ F358) doit être à 1.

		3.4.1	La tension en A6 TB1-54 doit être de 110 vdc.

		3.4.2	Vérifier le contact avant la borne A6 TB1-54.

		3.4.3	Remplacer la carte CF I/O si la tension est
			correcte.

4.0	Les relais C et CX doivent coller.

	4.1	La tension en A8 TB2-47 doit être de 110 vdc.
		Dans le cas contraire, tester:

		4.1.1	Le contact EES 4/7.
		
		4.1.2	Le contact MFS 11/12.

		4.1.3	Le contact ESC (vitesse supérieure ou
			égale à 6m/s) qui autorise suite à un arrêt
			d'urgence, un nouveau départ.

		4.1.4	Les contacts à rupture de 1M/2M (2CR/3CR)
			qui vérifient la retombée des contacteurs.
		
		4.1.5	Le contact BPS 9/10 qui vérifie la retombée
			du contacteur.
		
		4.1.6	Le contact H 8/7 qui vérifie la retombée
			du contacteur.

	4.2	Vérifier la bobine.

	4.3	Vérifier s'il existe un dur mécanique.

5.0	Le contacteur H doit collé.

	5.1	Vérifier le contact de C (suivant le schéma) qui 
		autorise les contacteurs H / BPS.

	5.2	Vérifier la diode H1 et la bobine.

	5.3	Vérifier s'il existe un dur mécanique.

	5.4	Remplacer la carte Relay I/O.

	5.5	Relais collé, le signal /H (@ F394) doit être à 0.

		5.5.1	La tension en A6 TB1-58 doit être de 0 vdc.

		5.5.2	Vérifier le contact avant la borne A6 TB1-54.

		5.5.3	Remplacer la carte CF I/O si la tension est 
			correcte.
	
6.0	Les contacteurs 1M / 2M doivent coller:	

	6.1	Se reporter aux défauts du Convertisseur Louis ALLIS:

		6.1.1	"Led RUN allumée sans les contacteurs".

Pour les Logiciels où le contrôle de BPS se passe en MLS4:

7.0	Le contacteur BPS doit collé.

	7.1	Vérifier:

		7.1.1	Les contacts de 1M / 2M ou 2CR / 3CR qui 
			permettent le collage de BPS.

		7.1.2	Les pares-flamme qui modifient parfois la 
			compression des doigts sur les contacts 
			de 1M / 2M.

		7.1.3	Les doigts de 1M / 2M qui peuvent se casser
			ou se fendre.

	7.2	Vérifier la bobine de BPS.

	7.3	Vérifier s'il existe un dur mécanique.

	7.4	Remplacer la carte Relay I/O.

	7.5	Relais collé, le signal BPS (@ F38A) doit être à 1.

		7.5.1	La tension en A6 TB1-53 doit être de 110 vdc.

		7.5.2	Vérifier le contact avant la borne A6 TB1-53.

		7.5.3	Remplacer la carte CF I/O si la tension est 
			correcte.
