----------------
Symptômes
----------------

1.0	Aucun défaut à l'OMT.
		
--------------------------
Causes Probables
--------------------------

1.0	Un relais du panneau A10 (DO / DO1 / DFO ...) est 
	défectueux.	

2.0	Résistances / Contacts de DOCB / OVL défectueux.
 
-------------
Remèdes
-------------
	  
1.0	Vérifier qu'en ouverture: 

	1.1	Les relais DO et DO1 soient collés.
		
		1.1.1	La tension de la bobine doit être égale à 
			110 vac et nulle lors d'une mesure en continu.

			1.1.1.1	L'interface ADO / NDO de la carte
				GP I/O peut être défectueuse.

			1.1.1.2	Les contacts de IDZ1 / IDZ2 / STOP
				autorisant DO / DO1 sont défectueux.

		1.1.2	Le relais peut être bloqué mécaniquement.

		1.1.3	Remplacer le relais si la tension est correcte.

	1.2	Les relais DFC, DFCX, DFO, NDG soient retombés:
		
		1.2.1	La tension de la bobine doit être égale à 0 vac.

		1.2.2	Le relais peut être bloqué mécaniquement.

		1.2.3	Remplacer le relais.

2.0	Vérifier que lors de la pleine ouverture: 

	2.1	Les relais DO, DO1, DFO soient collés.
		
		2.1.1	La tension de la bobine doit être égale à 
			110 vac	et nulle lors d'une mesure en continu.

			2.1.1.1	L'interface ADO / NDO de la carte
				GP I/O peut être défectueuse.

			2.1.1.2	Les contacts de IDZ1 / IDZ2 / STOP
				autorisant DO / DO1 sont défectueux.

		2.1.2	Le relais peut être bloqué mécaniquement.

		2.1.3	Remplacer le relais si la tension est correcte.

		2.1.4	Le contact DOL dans la DOCB doit être 
			actionné pour autoriser le relais DFO.

		2.1.5	Tester les 2 fils de cordon souple entre les 
			bornes:

				A8 TB1-33  >>  KJ1-90

				A8 TB1-7    >>  COP-HL1	
	
	2.2	Les relais DFC, DFCX, NDG soient retombés:
		
		2.2.1	La tension de la bobine doit être égale à 0 vac.

		2.2.2	Le relais peut être bloqué mécaniquement.

		2.2.3	Remplacer le relais.

3.0	Vérifier que lors de la fermeture: 

	3.1	Les relais DO, DO1, DFO soient retombés.
		
		3.1.1	La tension de la bobine doit être égale à 0 vac.

		3.1.2	Le relais peut être bloqué mécaniquement.

		3.1.3	Remplacer le relais.

	3.2	Les relais DFC, DFCX, soient retombés:
		
		3.2.1	La tension de la bobine doit être égale à 0 vac.

		3.2.2	Le relais peut être bloqué mécaniquement.

		3.2.3	Remplacer le relais.

	3.3	NDG ne doit coller que lors d'une fermeture forcée.

4.0	Vérifier:

	4.1	Les Résistances dans le contrôleur et la DOCB.

	4.2	Le pont redresseur RF2.

	4.3	Les contacts dans la DOCB.

	4.4	La tension d'excitation DMF et d'induit DMA.

	4.5	Les charbons sur le moteur de l'OVL.

	4.6	La fixation de la clavette sur l'arbre d'entraînement
		du moteur OVL.
