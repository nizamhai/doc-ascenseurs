FILE D'ATTENTE D'ERREUR
La file d'attente d'erreur est un utilitaire servant ŕ mémoriser les cinq derničres erreurs intervenues par ordre chronologique, pendant la marche de l'ascenseur Pour permettre d'entrer et de gérer la file d'attente d'erreur, vous devez appuyer sur la touche

MODE durant au moins 4 secondes pendant la visualisation de la position de cabine

Quand vous entrez ŕ l'intérieur de la file d'attente d'erreur un message "Err" s'affiche ŕ l'écran
Vous pouvez maintenant faire défiler la file d'attente de 5 erreurs avec les touches INC et DEC.Si vous soyez le symbole "-- --" pendant que vous faites défiler la file d'attente d'erreur au lieu d'un symbole d'erreur, cela signifie qu'il n'y a pas eu de génération d'erreur:
Si vous voulez revenir ŕ la visualisation de position de cabine, vérifier la file d'attente d'erreur jusqu'ŕ ce que vous voyiez ŕ l'écran le message "End", aprčs quoi, appuyez sur la touche MODE:
Chaque erreur est affichée avec un nombre et un symbole relatifs:
La premičre erreur s'affiche aprčs le message "Err", si on appuie sur la touche INC, on obtient la derničre erreur intervenue par ordre chronologique, c'est ŕ dire la plus récente. Pour réarmer la file d'attente d'erreur, il faut mettre hors fonction l'armoire de commande.
